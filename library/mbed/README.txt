Getting Started
---------------

(1) execute setup script to copy the source files to this directory and to make a zip file.

  (a) for Linux/UNIX
  $ ./setup.sh
  
  (b) for Windows
  > setup.bat
  or double-click the 'setup.bat' icon.

(2) create a new project in your mbed workspace for the sample code.

  - open your mbed online compiler.
  - click [New] button and create a new project in your workspace.
  - copy & paste the content of 'main.cpp' that is here.

(3) import the SerialScreen library to the project.

  - click [Import] button, and click [Upload] tab.
  - click [Brouze...] button, and select 'SerialScreen.zip' that is here.
  - click [Import!] button, and import it to the project.

(4) execute the sample code

  - click [Compile] button, and download the binary file.
  - copy the binary file to your mbed device.
  - push the reset button on the mbed device, and the sample code will start.
